# Spec Cheat Sheet
Cheat sheet for spec.dev

### Add a contract group to spec ecosystem
```bash
spec add contract group
```

### Add a contract to a contract group
```bash
spec add contract
```

### Run migrations
```bash
spec run migrations
```

### Run spec locally
```bash
spec start
```

### Run a local instance of graphql on spec
```bash
spec run graphql
```
